# Rotina App

Projeto de rotina diária desenvolvido para organizar atividades, registrar horários e facilitar o controle do tempo gasto em tarefas.  
Esse projeto foi criado como um sistema simples para praticar lógica de programação, manipulação de dados e estruturação de um aplicativo funcional.

---

## Sobre o Projeto

Este projeto permite ao usuário:
- registrar atividades com nome e horários de início e fim
- calcular automaticamente a duração de cada atividade
- armazenar informações em um banco simples
- visualizar toda a rotina do dia de forma organizada

Esse projeto foi desenvolvido com foco em praticar programação em Python, estruturação de código e lógica aplicada.

---

## Tecnologias Utilizadas

Este projeto usa as seguintes tecnologias:

- Python
- Manipulação de banco de dados simples (`rotina.db`)
- Scripts para automatizar ações (como `auto_git_push.py`)
- Estrutura modular com arquivos focados em lógica, banco e interface

---

## Estrutura do Repositório

Os principais arquivos e pastas são:

- `main.py`: arquivo principal que roda a aplicação
- `database.py`: funções para manipular o banco de dados
- `utils.py`: funções auxiliares para cálculo e validações
- `rotina_app/`: código da aplicação (lógica principal)
- `rotina.db`: banco de dados com registros de rotina
- `build/`, `dist/`: artefatos de build (se exportar a aplicação)
- `auto_git_push.py`: script para automatizar commits/push no Git
